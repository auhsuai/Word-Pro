# Word Pro - Luyện gõ tiếng Việt

Ứng dụng luyện gõ và kiểm tra tốc độ đánh máy tiếng Việt hiện đại.

## Tính năng
- **Kiểm tra tốc độ (Speed Test)**: Tính toán WPM (Từ mỗi phút) thời gian thực.
- **Độ chính xác (Accuracy)**: Phản hồi ngay lập tức lỗi chính tả bằng màu sắc.
- **Văn bản đa dạng**: Các đoạn văn bản tiếng Việt mẫu về nhiều chủ đề.
- **Giao diện hiện đại**: Sử dụng CustomTkinter với chế độ Dark Mode.

## Cách cài đặt
1. Cài đặt các thư viện cần thiết:
   ```bash
   pip install -r requirements.txt
   ```
2. Chạy ứng dụng:
   ```bash
   python main.py
   ```

## Hướng dẫn sử dụng
- Khi mở ứng dụng, một đoạn văn bản mẫu sẽ xuất hiện.
- Bắt đầu gõ vào ô nhập liệu bên dưới.
- Màu **Xanh** hiển thị ký tự đúng, màu **Đỏ** hiển thị ký tự sai.
- Nhấn "Làm mới" để thử đoạn văn bản khác.
